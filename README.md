# Cloudie
🌦️A modern, responsive, and feature-rich Weather Dashboard built with HTML, CSS, and JavaScript. This application provides real-time weather information using the **OpenWeatherMap API** with an intuitive UI, dynamic weather animations,hourly and 5-day forecasts, air quality monitoring, interactive charts, voice search, favorites, and much more.


## ✨ Key Features

* 🌍 Search weather by any city worldwide
* 📍 Get weather using your current location
* 🎤 Voice search with Speech Recognition
* 🌡️ Switch between Celsius (°C) and Fahrenheit (°F)
* 📅 Live date and real-time clock
* ⏰ Hourly weather forecast
* 📆 5-Day weather forecast
* 🌅 Sunrise & Sunset timings
* 🌤️ Dynamic weather icons and animations
* 📊 Temperature trend chart using Chart.js
* 🌬️ Wind speed & direction
* 💧 Humidity, Visibility, Pressure & Feels Like temperature
* 🌧️ Rain probability
* 🌿 Air Quality Index (AQI)
* 🗺️ Embedded Google Maps location
* ⭐ Save favorite cities
* 🕒 Recent search history
* 🌙 Dark/Light mode
* 📱 Fully responsive design for desktop, tablet, and mobile

## 🛠️ Tech Stack

* HTML5
